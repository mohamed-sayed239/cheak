Firebase Configuration

To use this demo, you need to set up a Firebase project and update the configuration below:

  1. Create a Firebase project at Firebase Console
  2. Enable Firestore Database
  3. Update the Firebase configuration in the code with your project details
Note: This is a demo for educational purposes. Please ensure you have permission to access cameras and microphones.
